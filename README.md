# Stock Taking

## Overview:
> This is a simple stock taking application that allows users to manage different types of stock. The app will have 3 main components: User sessions, Database management, and Stock management. 
> <br/>
> **Technologies**:
>  - Python 3.10.12
>  - Django 4.2.7
>  - MySQL
>  - JQuery - Ajax
>  - In-Place Editing
